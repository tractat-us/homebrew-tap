# homebrew-tap

Homebrew tap for [tractat-us](https://github.com/tractat-us) CLIs.

## Install

```sh
brew install tractat-us/tap/fgn
```

Or tap first:

```sh
brew tap tractat-us/tap
brew install fgn
```

## Formulae

| Formula | Description |
|---------|-------------|
| `fgn` | Fireworks Game Notation CLI — validate, format, and convert Hanabi game records |
